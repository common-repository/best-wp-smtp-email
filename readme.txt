Best and the Easiest SMTP Plugin for WordPress.

== Description ==
Best WP SMTP Email plugin is the most simple, easy and best SMTP emailer plugin for WordPress.

Using SMTP in your WordPress site was never been easier. Configuration is the easiest part of Best WP SMTP Plugin. Simply copy your SMTP credentials, and paste in plugin settings panel. That\'s it, no mumbo-jumbo!

**Features:**

* Beautiful Design
* Easiest Configuration Panel
* Supports WordPress Multisite
* Works Perfectly with Latest Version of WordPress
* Supports ALL SMTP Servers
* Supports SSL/TLS Security Protocol
* Supports No Authorization
* Built-in Usage Guide
* SMTP Test Email Option
* Works with ALL WordPress Themes
* Highly Secure

This plugin overrides the native wp_mail() function. So if you are having any issue with your current emails, this plugin will most probably solve that issue for you.

You will love this plugin for sure!

Please contact \"support@tinypixi.com\" if you face any kind of issue with this plugin. We will try our best to resolve.

== Installation ==
1. Upload the plugin zip file to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.

== Frequently Asked Questions ==
= Do I need to pay for using this? =
This plugin is absolutely FREE. However, your SMTP Server might charge you for their service. There are also free SMTP Server providers. You can try them.

= Will it fix my email issue? =
If the issue is with the native wp_mail() function, then in most cases it will fix the issue as it will override the wp_mail() function with SMTP Protocol.
